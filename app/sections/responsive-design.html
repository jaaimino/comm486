Responsive Design

Usage

Responsive design is a very big part of the modern web, embracing the mobile-first concept, rather than designing only for desktop use. Responsive design typically uses a grid system, in which elements are placed, and sized according to how much of a grid they are to take up. The web site then knows how to arrage these grid boxes to best fit different devices such as mobile phones, tablets, and even laptop computers.
